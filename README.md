# Build Project
1. Install Dependencies
    ```console
      npm install
    ```
3. Build the project
    ```console
      npm run build
    ```
4. Serve the project

   Using serve
    ```console
      serve dist
    ```
   Using http-server
    ```console
      http-server dist
    ```

# Run the Project for Development
1. Install Dependencies
    ```console
      npm install
    ```
2. Run the project
    ```console
      npm run dev
    ```
